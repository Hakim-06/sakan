1. Resume Executif

SakanCampus est une plateforme web complete centralisant annonces immobilieres, matching intelligent, messagerie P2P et assistant IA. Elle facilite la recherche logement etudiant au Maroc, reduit le temps de prise de decision et ameliore la qualite des mises en relation.

4. Architecture Globale

Client (React/Vite) → HTTPS ← Backend (Express/Node) ← MongoDB. Services externes: Cloudinary, OpenAI, Gemini, Nodemailer.

5. Modules Fonctionnels

🔐 Authentication

  • Inscription email+password, validation, hash bcrypt
  • Connexion local ou Google OAuth (verif backend)
  • OTP email 6-chiffres pour verify/reset
  • JWT token generation & validation

👤 Profil Utilisateur

  • Infos: nom, age, genre, ecole, ville, budget
  • Photo upload → Cloudinary 300x300
  • 12 traits selectionables
  • Bio generation IA (Gemini)

🏠 Annonces

  • CRUD complet (owner-only edit/delete)
  • 10 amenities (Wi-Fi, Clim, Parking, etc.)
  • Multi-photos upload Cloudinary

🔀 Matching Algorithm

  • Score = ville(30pts) + budget(proximity) + traits(Jaccard)
  • Seuil configurable 0-100%

💬 Messagerie

  • Conversations P2P, unread per conversation
  • Sent/seen markers, pinned, archive

🤖 Assistant IA

  • POST /api/ai/chat: question + context → OpenAI/Gemini/Local
  • Mirroring langue (Darija/FR)
  • Greeting short-circuit (salam = 1 ligne)
  • Provider fallback avec timeouts 20s

{_id, owner:UserId, title, description, city, budget, school, amenities[10], images[{url,publicId}], createdAt, updatedAt}

8. IA Provider Fallback Flow

User Question ↓ Validation + Context Build (top cities, prices, user profile) ↓ Try Provider (OpenAI/Gemini per AI_PROVIDER env) ↓ Pipeline: [OpenAI: GPT-4o, 4o-mini, 4.1-mini] → [Gemini: 2.5-flash, 2.0-flash, 1.5-flash] ↓ Each model: timeout 20s, retry 1x on fail ↓ First SUCCESS → return {text, mode: "openai:model" | "gemini:model"} ↓ All FAIL → buildLocalChatFallback() ↓ Response + label "IA" or "IA (fallback)" or "Local"

9. Securite & Fiabilite

🛡️ Mesures Securite

  • JWT auth stateless, expiration configurable
  • Bcryptjs hashing (round 10+)
  • Helmet: CSP, X-Frame-Options, no-sniff, hsts
  • CORS whitelist frontend origins
  • Rate-limit: 100 req/15min per IP
  • Express Validator: sanitize + validate inputs
  • No SQL injection (Mongoose schemas)
  • No XSS (React auto-escape)

🔄 Fallbacks & Fiabilite

  • IA provider orchestration (OpenAI→Gemini→Local)
  • Timeout enforcement 20s global per call
  • Retry logic multi-model au sein provider
  • Error handling + graceful degradation
  • Local fallback reponses intelligentes
⚠️ CRITIQUE: API keys ont ete exposees. Action immédiate: regenerer TOUTES les clés (OpenAI, Gemini, Cloudinary, MongoDB URI, Google OAuth).
