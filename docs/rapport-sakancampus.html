1. Introduction

SakanCampus est une plateforme web concue pour repondre a un besoin concret: faciliter la recherche de logement et de colocation pour les etudiants au Maroc. Le projet centralise les annonces, la communication entre utilisateurs, la gestion de profil et des fonctions intelligentes d'aide a la decision.

L'objectif principal est d'offrir une experience moderne, fluide et fiable sur ordinateur et mobile, tout en reduisant les frictions habituelles: manque d'information claire, difficultes de matching, lenteur des echanges et absence d'un espace unifie.

3. Objectifs de la plateforme

  • Faciliter la publication et la consultation d'annonces de logement.
  • Permettre un matching plus pertinent via profil, budget, ville et preferences.
  • Offrir une messagerie integree pour accelerer la prise de contact.
  • Garantir une experience responsive, rapide et exploitable sur mobile.
  • Integrer une assistance IA pour guider les utilisateurs dans leurs choix.

4. Technologies utilisees

Frontend
  • React
  • Vite
  • React Router
  • CSS custom + design responsive
Backend
  • Node.js
  • Express.js
  • JWT (authentification)
  • Nodemailer (emails)
Data & Services
  • MongoDB + Mongoose
  • Cloudinary (images)
  • OpenAI / Gemini (assistant IA)
Deploiement
  • Vercel (frontend + API route)
  • Configuration env vars
  • Build CI simple via GitHub

5. Architecture generale

5.1 Couche presentation (Frontend)

L'interface est construite autour de pages React (Login, Profil, Feed, VerifyEmail), avec navigation dynamique, composants reutilisables et interactions optimises pour le tactile.

5.2 Couche logique (Backend API)

L'API Express gere l'authentification, les annonces, les messages, les uploads et les operations IA. Les routes sont proteges via JWT selon le role et l'etat de connexion.

5.3 Couche donnees

MongoDB stocke les utilisateurs, annonces et conversations. Les schemas Mongoose imposent la structure, les validations et les defaults necessaires a la coherence de la plateforme.

6. Fonctionnalites principales

  • Inscription / connexion classique + connexion Google.
  • Verification email par code OTP.
  • Reset mot de passe par code OTP.
  • Gestion complete du profil utilisateur.
  • Publication d'annonce avec photos.
  • Favoris persistants et matching filtre.
  • Messagerie interne avec etat de lecture.
  • Assistant IA contextuel (prix, villes, filtres, conseils).

7. Securite et fiabilite

  • Authentification securisee via JWT.
  • Endpoints sensibles proteges par middleware auth.
  • Fallback IA pour garantir une reponse meme en cas d'echec provider.
  • Gestion des erreurs UX pour eviter les blocages utilisateur.

8. Conclusion

SakanCampus est un projet full-stack complet, orientee impact utilisateur, qui combine experience mobile moderne, outils de mise en relation et assistance intelligente. La plateforme est exploitable en production et evolutive vers des fonctionnalites avancees (temps reel, recommandations plus fines, analytics, etc.).
